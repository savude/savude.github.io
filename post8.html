My Experience with JavaScript

Posted by Ellen Savude, on July 12, 2018

JavaScript is a client-side programming language, which means it’s executed in the user’s web browser. In principle, JavaScript is a very dynamic and interactive programming language. However, modern day JS is a force to reckon with its extensive build and compilation steps. For rookies like me, it’s made my life miserable, because why does something truly have to be so complicated? But it also has been worthwhile, now that I can understand a thing or two about how to write it, effectively (I really do hope that’s true for the latter).

For this uphill task to learn this new language, I can’t be grateful enough to Khan Academy, for providing me with insightful, interactive and practical material to introduce me to the language. I went through great lessons, practice quizzes and projects which allowed me to test & apply my knowledge on the concepts I learned through the different modules. Another great resource for me was Free Code Camp on GitHub. I haven’t finished their 6 billion lessons, but the resources there were impeccable. They started right from the beginning of HTML, them CSS then JS which I thought was really awesome, because it allowed me to incorporate all these languages in a manner that makes sense (I hope) and in a beautiful, responsive format. I’ll continue with my lessons on Free Code Camp, because as I said previously, they are extensive, but for now I’m super glad that they were there through my sleepless nights of trying to crack JS.
